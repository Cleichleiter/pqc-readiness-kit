Samples

This directory contains sanitized example outputs to help reviewers understand the data model and report artifacts without running the toolkit.

sample_outputs

crypto_inventory.sample.json
Example structure produced by the Windows inventory workflow. Fields are representative and values are non-sensitive placeholders.

findings.sample.csv
Example findings backlog showing typical hygiene and PQC-relevance signals. This format is designed to be sortable and easy to ingest into ticketing or planning tools.

These sample files are intended for documentation and demonstration only and do not represent real scan results.
